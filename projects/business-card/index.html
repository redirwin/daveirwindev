About

The objective of this project was to create a simple business-card website that is responsive to large and small screen sizes.

In keeping with the simple concept, I chose a plain color scheme that uses a dark-blue accent and text color, implementing a darker shade of blue based on the prominent blue in the photograph.

This project was completed using only HTML and CSS.
